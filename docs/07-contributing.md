# Contributing

How to contribute
- Fork or branch for personal variants. Use branch-per-person for customizations.
- Create PRs to propose shared/common changes.
- Use issues for TODOs and feature requests.

Suggested workflow
- Branch name: username/feature.
- Migrate content into docs/ and open a PR with rationale.
- Keep personal variants in separate branches and consider a manifest of differences.
